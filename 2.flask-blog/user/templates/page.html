Python -Flask推荐学习配置

发布时间：
17-08-18
文章分类：
python


环境：Ubuntu(17.04)版
Python版本：内置的Python3.5.3
IDE：Pycharm,Sublime text 3




学习Python推荐使用Linux环境..主要是Linux能通过依赖安装解决大部分的安装问题
减少了学习的成本和难度。所以之后的教程是在Linux下完成的
（我是用VM虚拟机搭建的Linux,想学习的同学也可以用VM搭建熟悉后再找一台旧电脑装上）




 用的是Python3,但是Ubuntu17.04也有Python2,所以在调试过程中一定要输入Python3 *
首先安装好编写环境，先打开终端（Ctrl+Alt+T）输入以下命令： 


$ sudo apt-get install python3-pip

然后需要用到虚拟环境，我们使用Py内置的venv（Python3.3之后内置的） 
$ sudo apt-get install python3-venv

进入项目地址 通过 
sudo mkdir venv

新建目录并且执行以下命令 
pyvenv venv

通过以下命令进行虚拟环境（如果不是使用Linux系统则无法用到以下命令） 
. venv/bin/activate

安装好后再通过以下命令安装Flask： 
$ pip3 install Flask

安装好后进入Python环境,写入: 
(venv)$ Python3

>>> import flask

按回车后没报错就安装完成了。
